<p>Follow these steps to connect your Trackensure account to TruckerCloud:</p>
<ol>
  <li>Send an email to <a href="mailto:acc.manager@trackensure.com">acc.manager@trackensure.com</a> to request an API key</li>
  <li>When you receive the API key, return to the TruckerCloud onboarding widget</li>
  <li>Paste the API key and click <strong>Verify Credentials</strong>, then <strong>Complete Sign Up</strong></li>
</ol>
