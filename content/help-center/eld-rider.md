<p>Follow these steps to connect your ELD Rider account to TruckerCloud:</p>
<ol>
  <li>Log in to your ELD Rider account at <a href="https://eldrider.com/#/auth/login">https://eldrider.com/#/auth/login</a></li>
  <li>Scroll down the left menu and click <strong>Admin</strong></li>
  <li>From the dropdown menu select <strong>API</strong></li>
  <li>In the top right corner select <strong>+ Add API Key</strong></li>
  <li>Name the API <strong>Insurance</strong></li>
  <li>Select <strong>Read-Only</strong> and click <strong>Confirm</strong></li>
  <li>Copy the generated API key</li>
  <li>Return to the TruckerCloud onboarding widget and paste the key</li>
  <li>Click <strong>Verify Credentials</strong>, then <strong>Done</strong></li>
</ol>
