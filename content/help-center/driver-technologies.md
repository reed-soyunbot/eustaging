<p>Follow these steps to connect your Driver Technologies account to TruckerCloud:</p>
<ol>
  <li>When you signed up for Driver, you should have received an email with your API username, password, and API Key</li>
  <li>If you cannot find that email, send a message to <a href="mailto:support@drivertechnologies.com">support@drivertechnologies.com</a> CC'ing <a href="mailto:support@truckercloud.com">support@truckercloud.com</a> with the subject <strong>TruckerCloud API</strong> asking for API information</li>
  <li>Copy the username, password, and API Key</li>
  <li>Return to the TruckerCloud onboarding widget and click <strong>Authorize</strong></li>
  <li>Paste the values in the fields</li>
  <li>Click <strong>Verify Credentials</strong>, then <strong>Done</strong></li>
</ol>
