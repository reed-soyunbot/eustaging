<p>Follow these steps to connect your Teletrac Navman account to TruckerCloud:</p>
<ol>
  <li>Email <a href="mailto:integrations@teletracnavman.com">integrations@teletracnavman.com</a> and CC <a href="mailto:support@truckercloud.com">support@truckercloud.com</a></li>
  <li>Ask Teletrac to send you an API Key</li>
  <li>Once you receive the API Key, return to the TruckerCloud onboarding widget</li>
  <li>Click <strong>Authorize</strong> and paste your API Key</li>
  <li>Click <strong>Verify Credentials</strong>, then <strong>Done</strong></li>
</ol>
