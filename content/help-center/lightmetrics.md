---

<p>Lightmetrics requires a data sharing agreement before connecting. Follow these steps:</p>
<ol>
  <li>Complete the <a href="https://na4.docusign.net/Member/PowerFormSigning.aspx?PowerFormId=98f0bc71-21ae-45b2-beb6-ad4e9f74556f&env=na4&acct=dcc58713-002d-4605-86a9-9be5e3f4bf55&v=2">Lightmetrics DocuSign agreement</a></li>
  <li>Lightmetrics will receive a copy and create your API credentials</li>
  <li>Your API credentials will be sent to you via email</li>
  <li>Once you receive credentials, return to the TruckerCloud onboarding widget</li>
  <li>Click <strong>Authorize</strong> and paste the credentials in the relevant fields</li>
  <li>Click <strong>Verify Credentials</strong>, then <strong>Done</strong></li>
</ol>
