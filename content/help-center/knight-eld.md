<p>Follow these steps to connect your Knight ELD account to TruckerCloud:</p>
<ol>
  <li>Email <a href="mailto:admin@knighteld.com">admin@knighteld.com</a> to request an API key for TruckerCloud</li>
  <li>Once you receive the API key, return to the TruckerCloud onboarding widget</li>
  <li>Click <strong>Authorize</strong></li>
  <li>Paste the API key, then click <strong>Verify Credentials</strong>, then <strong>Done</strong></li>
</ol>
