<p>Follow these steps to connect your Raven account to TruckerCloud:</p>
<ol>
  <li>Send an email to <a href="mailto:support@ravenconnected.com">support@ravenconnected.com</a> requesting an API key and Secret, CC'ing <a href="mailto:support@truckercloud.com">support@truckercloud.com</a></li>
  <li>Once you receive the information, return to the TruckerCloud onboarding widget</li>
  <li>Click <strong>Authorize</strong></li>
  <li>Copy and paste the API Key and Secret into the fields</li>
  <li>Click <strong>Verify Credentials</strong>, then <strong>Done</strong></li>
</ol>
