<p>The Geographic Risk dashboard shows where your fleet's vehicles are driving — broken down by zip code, state, and risk level. Helps underwriters and actuaries understand territorial exposure based on actual travel patterns.</p>
<h2>Heatmap Visualization</h2>
<p>Interactive heatmap showing miles driven by zip code. Darker tiles = more miles. Covers the US, Canada, and Mexico. Zoom in for granular zip code detail.</p>
<h2>Territory Risk Factor Map</h2>
<p>A separate heatmap showing risk rating factors by zip code. Darker red = higher risk zones. High-risk examples: downtown Atlanta, Southern California, South Florida, Louisiana. Lower-risk: rural regions like South Dakota.</p>
<h2>Miles by Zip Code Table</h2>
<p>Detailed table exportable to Excel. Actuaries can apply their own rating factors to actual miles by zip code to re-rate fleets based on real geographic exposure.</p>
<h2>Canada &amp; Mexico</h2>
<p>Canada uses Forward Sortation Areas (FSAs). Mexico uses zip code equivalents. A country code differentiates US/Mexico since they share some zip formats.</p>
