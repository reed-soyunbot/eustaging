<h2>Steps to Connect</h2>
<ol><li>Email support@swiftteld.com and request an API key for TruckerCloud.</li><li>Once you receive the API key, return to the TruckerCloud onboarding page.</li><li>Click <b>Authorize</b>, paste the API key, and click <b>Verify Credentials</b>.</li><li>Click <b>Done</b>.</li></ol>
