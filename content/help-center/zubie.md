<p>Follow these steps to connect your Zubie account to TruckerCloud:</p>
<ol>
  <li>Send an email to <a href="mailto:support@zubie.com">support@zubie.com</a> CC'ing <a href="mailto:support@truckercloud.com">support@truckercloud.com</a> with subject <strong>API Key</strong></li>
  <li>Ask Zubie to generate an API key for you</li>
  <li>Once you receive the API key, return to the TruckerCloud onboarding widget and click <strong>Authorize</strong></li>
  <li>Paste the API key in the field</li>
  <li>Click <strong>Verify Credentials</strong>, then <strong>Done</strong></li>
</ol>
