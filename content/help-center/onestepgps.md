<p>Follow these steps to connect your OneStepGPS account to TruckerCloud:</p>
<ol>
  <li>Send an email to <a href="mailto:integrations@onestepgps.com">integrations@onestepgps.com</a> with subject line <strong>TruckerCloud</strong>, CC'ing <a href="mailto:support@truckercloud.com">support@truckercloud.com</a></li>
  <li>Ask OneStepGPS to generate an API key for you</li>
  <li>Once you receive the API key, return to the TruckerCloud onboarding widget</li>
  <li>Click <strong>Verify Credentials</strong>, then <strong>Done</strong></li>
</ol>
