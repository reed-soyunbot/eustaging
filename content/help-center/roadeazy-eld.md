<h2>Steps to Connect</h2>
<ol><li>Log into your RoadEazy account at https://app.roadeazy.com/</li><li>Navigate to <b>Settings → API Keys</b></li><li>Click <b>+ Add API Key</b>, name it, extend expiration as far as possible, and select all roles</li><li>Copy the API key</li><li>Return to the TruckerCloud onboarding link and click the RoadEazy tile</li><li>Click <b>Authorize</b>, paste the key, and click <b>Verify Credentials</b></li><li>Click <b>Done</b></li></ol>
